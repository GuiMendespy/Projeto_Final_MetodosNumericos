## Projetos Finais de Métodos Numéricos

# Sistemas de Recomendação:

No qual vocês terão de entender de como funcionam e como construir sistemas de recomendação e do uso de fatorização matricial nesse processo, em especial, usando Decomposição de Valor Singular (SVD) e construir uma aplicação direta desse problema. Referências: curso Building Recommendation Engines in Python do Datacamp, tutoriais Build a Recommendation Engine With Collaborative Filtering e Singular Value Decomposition – A Comprehensive guide on Singular Value Decomposition, com link em anexo.

# Modelagem de Tópicos:
No qual vocês entenderão como agrupar, transformar, visualizar e extrair insights de conjuntos de dados não rotulados, expressando documentos como combinações de tópicos e imagens em termos de padrões visuais comuns, usando uma variedade de técnicas em aprendizagem de máquina, desde agrupamento até redução de dimensão e fatoração de matrizes, em especial, Fatoração de Matrizes Não-Negativas (NMF). Referências: curso Aprendizado não supervisionado em Python no Datacamp e os tutoriais Topic Modeling with NMF and SVD, NMF - Non-Negative Matrix Factorization, What is Topic Modeling? An Introduction With Examples, com link em anexo.

# Remoção de plano de fundo de uma imagem:
No qual vocês irão usar técnicas de fatoração matricial para separar elementos do primeiro plano de elementos do segundo plano de uma imagem, usando, principalmente Decomposição de Valor Singular (SVD). Referência: tutorial Background Removal with SVD, Computational Linear Algebra: Background Removal with Principle Component Analysis, com link em anexo.
